Interaction and navigation patterns in Metaverse scenario

WebXR Wiki, ASG Avanade

Project introduction

The goal is to have an overview of exsisting interaction and navigation patterns, with examples to be able to test them with a hands on experience. The application will be a fully immersive wiki, developed for the Meta (Oculus) Quest 2, which is a virtual reality headset developed by Meta (formerly known as Facebook), and it should be easily acessible. The application will work as a demonstration to help with the decision making of relevant patterns and will function as a base application for future projects.

Interaction patterns

What are Interaction Patterns

They are the interactions methods that can be used to interact with the virtual environment. For example, different ways of picking up a cube and resizing it. With the help of sensor and controller information, it is possible to try and make the movements as natural as currently possible.

Navigation patterns

What are Navigation Patterns

In XR, navigation patterns refer to the various methods by which a user can navigate within a virtual environment, including techniques for menu navigation (moving from one scene to another) and user movement within a scene. For this project we will be looking at the later one, the methods a user can maneuver through a scene.
