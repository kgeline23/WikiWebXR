Navigation Patterns in Extended Reality

Posted by Klaudia Konadu-Finke, ASG Avanade on December 22, 2022

What are Navigation patterns

Just like in other digital products navigation is an important aspect, and crucial to produce an immersive and accessible environment. In XR, navigation patterns refer to the various methods by which a user can navigate within a virtual environment, including techniques for menu navigation (moving from one scene to another) and user movement within a scene. For this project we will be looking at the later one, the methods a user can maneuver through a scene.

It is important to make a thorough decision when choosing a navigation pattern, as it is directly connected to the user’s experience. Depending on the pattern and the individual user there might be a high chance of motion sickness, to try and minimize this it is important to continuously test the pattern and make relevant adjustments. The section below describes the possible patterns in XR.

The types of Navigation Patterns

Viewpoint Control Patterns

Physical Navigation (Locomotion)

Using the tracking of the user’s physical body, to move within the virtual world.


Virtual Navigation

This uses the help of controllers or joystick to move within the scene without having to move their physical body. If not done properly it can lead to motion sickness.


Steering

Changing users view by tracking the users head movement or using an external controller, however using a controller can lead to motion sickness if it is not tested intensively.


ADD Patterns

3D multi-touch

It is used to navigate through the environment by, for example, dragging the hand to change the rotation, zooming to change scale of it and steering to changing location.

Teleportation

Jumping from one hotspot to another, without physically moving but rather selecting the next position with a pointer or pressing a button on the controller to move.


Scene Navigation

Stationary scene

In this the precision tracking is not used therefore the user is not able to roam the scene but is placed and fixed in one spot, so when moving from one scene to another it uses teleportation. This can also be done by using a menu to navigate.


Free moving scene

This allows the user to freely walk around the scene by using the above-mentioned viewpoint controls.
